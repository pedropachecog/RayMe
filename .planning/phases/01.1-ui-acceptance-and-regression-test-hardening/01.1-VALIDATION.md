## Sampling Rate

- After every task commit: run the narrow E2E/API command for the touched surface.
- After every client-code task commit: also run `npm --prefix web-ui/client run test:unit -- --run`.
- After every plan wave: run `uv run --project web-ui/server pytest web-ui/server/tests -q && npm --prefix web-ui/client run test:unit -- --run && npm --prefix web-ui/client run test:e2e`.
- Before `$gsd-verify-work`: full local suite must be green, live `OMEN-PC` browser smoke must be green with console/page-error capture, then product-owner Android Chrome checkpoint may be requested.
- Max feedback latency: no three consecutive implementation tasks may land without an automated verification command.
