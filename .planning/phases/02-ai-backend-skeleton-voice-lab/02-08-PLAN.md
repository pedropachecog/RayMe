## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Web UI server to `/tts/synthesize` | Reference audio, transcript, and synthesis text enter engine adapters. |
| TTS adapter to GPU runtime | Engine load/switch can consume VRAM and fail per adapter. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-08-01 | Denial of Service | Engine switching | mitigate | One resident engine, unload-before-load, per-engine unavailable state. |
| T-02-08-02 | Information Disclosure | TTS errors | mitigate | Return fixed `Synthesis failed` public errors without local paths or tracebacks. |
| T-02-08-03 | Repudiation | Engine caveats/licenses | mitigate | Registry metadata includes license, caveats, and runtime evidence for all six engines. |
| T-02-08-04 | Tampering | Durable state boundary | mitigate | Tests forbid AI backend voice library/list persistence endpoints. |
</threat_model>
