<objective>
Contain VoxCPM2 native runtime crashes and hangs so a single failed Rehearsing turn cannot kill the RayMe AI backend or block future call startup.

Output: VoxCPM2 production synthesis/streaming runs in a supervised CUDA-only worker process; worker death or timeout becomes a sanitized recoverable failure; streamed chunks still reach the parent as they are generated without whole-synthesis fallback.
</objective>

<user_goal_preservation>
The user must be able to start another RayMe call after a VoxCPM2 generation failure. The live-call path must remain GPU-only, streaming, bounded, and recoverable.
</user_goal_preservation>
