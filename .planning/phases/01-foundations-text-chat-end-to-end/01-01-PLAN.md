## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| card file -> backend parser | Untrusted JSON/PNG bytes cross into RayMe storage contracts. |
| browser -> API | Unauthenticated LAN client requests reach app APIs. |
| backend -> LLM endpoint | Server-side proxy uses sensitive endpoint/key settings. |
| migration -> SQLite | Durable schema decisions can cause future data loss if underspecified. |
