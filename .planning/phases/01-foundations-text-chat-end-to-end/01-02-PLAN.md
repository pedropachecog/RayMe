|----------|-------------|

| browser app -> API | SPA calls unauthenticated LAN APIs. |
| browser viewport -> UI contract | Mobile and desktop layouts must not hide unsafe/future controls. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-02-XSS | E/T | `renderMarkdown.ts` | mitigate | Use Marked plus DOMPurify with explicit allowed tags/attrs and malicious literal tests. |
| T-01-02-KEY | I | Settings/E2E contracts | mitigate | E2E shells require masked key behavior and no key echo in UI text. |
| T-01-02-FUTURE | T | App shell | mitigate | Static grep blocks future Voice Lab/Call/account/billing/logout controls in Phase 1. |
| T-01-02-MOBILE | D | Mobile viewport specs | mitigate | Playwright mobile spec requires import/chat/reload path to remain reachable without overlap. |
